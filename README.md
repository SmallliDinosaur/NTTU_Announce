# NTTU_Announce
爬蟲學校公告並通知
<br>每小時爬取六種公告五則並覆蓋<br>
每天八點有每日二十單更新<br>
優點：可直接在搜尋欄搜尋過去的公告<br>
請將網頁以 Chrome 開啟<br>
**建議使用無痕或手機掃描**
